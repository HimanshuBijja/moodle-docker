My Dashboard Plugin for Moodle
==============================

This plugin adds a "My Dashboard" navigation item to Moodle that allows users to save and manage their favorite links.

Features:
- Add, edit, and delete custom links
- Display links as responsive cards (2 per row)
- Clean, modern interface
- User-specific link storage
- No special capabilities required (all users can use it)

Installation:
1. Place the 'mydashboard' folder in your Moodle's 'local' directory
2. Visit Site Administration > Notifications to install the plugin
3. The "My Dashboard" link will appear in the main navigation menu

Usage:
- Navigate to "My Dashboard" from the main menu
- Click "Add New Link" to create a new link
- Fill in the title, URL, and optional description
- Manage your links using the edit/delete buttons

The plugin creates a table 'local_mydashboard_links' to store user links.
