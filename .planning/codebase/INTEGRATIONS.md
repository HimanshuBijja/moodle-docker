# Integrations

## Authentication & Identity (moodle/auth/)
- **CAS**: Central Authentication Service.
- **LDAP**: Lightweight Directory Access Protocol.
- **LTI**: Learning Tools Interoperability (as both consumer and provider).
- **OAuth2**: Supports Google, Microsoft (Azure AD), Facebook, and custom providers.
- **Shibboleth**: Federative authentication.
- **MNet**: Moodle Network (cross-Moodle authentication).

## Storage & Repositories (moodle/repository/)
- **Cloud Storage**: Dropbox, Google Drive, OneDrive, Amazon S3, Nextcloud.
- **Media**: Flickr, YouTube, Wikimedia.
- **Protocols**: WebDAV, File System, Equella.

## Enrolment Methods (moodle/enrol/)
- **Manual**: Manual user enrolment.
- **Self**: Student-initiated enrolment.
- **Payment**: PayPal, Fee-based enrolment.
- **External Systems**: IMS Enterprise, Database, LDAP.

## Messaging & Communication
- **AI Providers**: Core support for AI subsystems (AI placement and providers).
- **Email**: SMTP and various mailers.
- **SMS**: Integration via SMS subsystem (AWS SNS and others).

## Content & External Tools
- **H5P**: Interactive content framework.
- **LTI (External Tool)**: Integration with external learning tools.

## Other Services
- **AWS**: Bundled AWS SDK indicates potential for S3, SNS, and other AWS services.
- **Brickfield Accessibility Toolkit**: Accessibility checking service.
- **MathJax/TeX**: For rendering mathematical notations.
- **MaxMind**: For GeoIP lookups (IP lookup subsystem).
